existirá um usuário admin que poderá controlar tudo.

As pessoas poderão se registrar e autenticar para usar o site.

O usuário admin precisa permitir que um usuário acesse um (ou mais) site específico.

O usuário pode acessar o site após ser permitido pelo admin.
O usuário que tem a permissão de colocar 'sub usuários' para um site específico.

Ex.
Admin permite uma pessoa a qualquer site, pois ele tem acesso a todos.
Usuário permitido apenas para o site que ele tem acesso.
O Admin pode escolher não dar permissão de adicionar usuários para um usuário.

As permissões se baseam em:
- Site em questão (usuário x site)
- Operação (Gerir produtos, gerir usuários, ver métricas, etc.)
